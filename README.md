## CasaWood Website

This is the codebase for the CasaWood website, built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

### Getting started

1. Install dependencies:

```sh
npm install
```

2. Run the development server:

```sh
npm run dev
```

3. Build for production:

```sh
npm run build
```

4. Preview the production build:

```sh
npm run preview
```

You can edit the source under `src/` and the API routes under `api/`.
